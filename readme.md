Simple CRUD REST API Laravel 5.8
============

##Prerequisites
- Laravel >= 5.8


## Basic Usage

in the terminal Run this command,
```
composer update

```
create file .env
```
cp .env.examples .env

```

generate key
```
php artisan key:generate

```
run
```
php artisan serve

```

and open this URL in your browser

```
http://localhost:8000/api/siswa

```
